# 贵金属多尺度风险溢出研究

## 01. 数据探索

在第一阶段探索性分析之后，发现四种贵金属现货、期货数据涵盖的日期是：2012-01-03 到 2026-03-31。一共有 3101 个共同日期，计算收益率，收益率计算公式是：$100 \cdot (\log P_t-\log P_{t-1})$。

precious_metals_price_panel_complete.csv 文件是将4种贵金属的现货与期货价格进行汇总，其中期货价格都来自类似于 xagusd.csv 文件，他们都是记录的贵金属的现货价格，我们使用的是 close 列，期货价格都来自与 Historical Data 结尾的文件记录的是期货价格，我们使用的是 Price 列。 4种贵金属的期货与现货价格显然有相关性，所以我们分别计算 2012-01-03 到 2026-03-31 的日收益率相关系数，得到的是：

```
,futures_spot_return_corr,pair_overlap_rows
gold,0.8541294438428556,3100.0
silver,0.8680878070523738,3100.0
platinum,0.926570365574015,3100.0
palladium,0.944448187786282,3100.0
```

对四种贵金属期货现货价格的收益率计算关键的统计量，详细结果在precious_metals_return_descriptive_stats.csv

- 收益率均值都很小

> 日均收益大约在 0.01% 到 0.035% 之间，符合金融资产日收益的一般特征：均值很小，波动才是主要信息。

- 期货波动普遍高于现货

标准差上，四种金属都是期货略高于对应现货：

黄金：期货 1.128，现货 1.085
白银：期货 2.222，现货 2.093
铂金：期货 1.848，现货 1.737
钯金：期货 2.446，现货 2.306
这支持后续研究中的一个基本判断：期货市场可能是更敏感的风险反应端。

- 波动强度排序大致是：钯金、白银、铂金、黄金

> 黄金最稳定；钯金和白银波动明显更大。这和它们的市场属性一致：黄金避险和货币属性更强，白银、铂金、钯金工业属性更强，受供需冲击影响更明显。

- 所有序列都是负偏态

> skew 全部小于 0，说明样本里大跌风险比大涨风险更突出。白银现货和白银期货负偏尤其明显。

- 所有序列都尖峰厚尾

> 正态分布峰度约为 3，但这里所有序列峰度都远大于 3。白银期货峰度达到 36.86，白银现货达到 33.99，说明极端波动非常明显。

- Jarque-Bera 检验拒绝正态性

> 所有 jb_pvalue = 0，说明收益率不服从正态分布。这为后续使用 ICEEMDAN 这类适合非线性、非平稳金融波动结构的方法提供了理由。

- ADF 检验显示收益率平稳

> 所有 adf_pvalue 都接近 0，说明收益率序列可以认为是平稳的。这个结果很重要，因为后续做分解、连通性分析时，收益率比价格水平更适合作为输入。
